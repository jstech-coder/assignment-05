Question-1: What are the different ways to select an element in the DOM?

There are 5 ways in which you can select elements in a DOM using selectors.

getElementsByClassName()

querySelector()
querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

both innerText and textContent will ignore the HTML tags and render everything as a string. while textContents is all text contained by an element and all its children

Question-3: What is event delegation in the DOM?

Event Delegation is basically a pattern to handle events efficiently.

Question-4: What is event bubbling in the DOM?

Event bubbling allows a single event listener on a parent element to handle events for multiple child elements, making event management more efficient.
